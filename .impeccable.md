# Couponawy — Design Context

## Design Context

### Users

Arabic-speaking online shoppers across the Gulf and MENA region. They are deal-seekers — motivated by savings, driven by discovery. They arrive with intent: "I want to buy something and I want the best price." Their context is mobile-first, Arabic-native, and time-sensitive. The job to be done is: **find a valid coupon fast, feel the win, move on to checkout**.

### Brand Personality

**Bold. Generous. Alive.**

Couponawy should feel like the friend who always knows the deal — enthusiastic, dependable, never boring. The emotional goal is a small burst of delight on every visit: the feeling of getting away with something good.

Anti-personality: clinical, corporate, sleepy, overly minimal, purple-tech-startup.

### Aesthetic Direction

**Vibrant + Editorial Arabic**

- **Visual energy**: Groupon/RetailMeNot boldness — strong color, clear hierarchy, promotional without being chaotic
- **Color system**: Token names are `brand-red` (crimson-coral primary) and `brand-gold` (amber accent).
  - **Primary** (`brand-red`): Bold crimson-coral — energy, urgency, deals
  - **Accent** (`brand-gold`): Warm amber/gold — treasure, value, celebration
  - **Surface** (`cream` / `cream-dark`): Warm white — clean but not sterile
  - **Text** (`charcoal`): Deep warm charcoal — readable, grounded
- **Light mode only** — no dark mode needed now
- **Reference**: RetailMeNot/Groupon energy, but designed for Arabic reading patterns and cultural context
- **Anti-reference**: Purple tech startups, generic SaaS dashboards, cold minimal white

### Design Principles

1. **The deal is always the hero.** Discount percentages, coupon codes, and savings amounts get the biggest visual weight. Everything else is support.

2. **Urgency without pressure.** Bold doesn't mean loud. Use strong color and clear hierarchy to create momentum — not anxiety. No fake countdown timers. No red everywhere.

3. **Arabic-first, not Arabic-translated.** RTL layout is a first-class citizen. Typography breathes. Noto Kufi Arabic for display, Tajawal for body. Spacing and rhythm calibrated for Arabic text density.

4. **Scan in 3 seconds.** A user should be able to identify the best coupon on any page in under 3 seconds. Grid layouts, bold headlines, clear category signals.

5. **Trust through clarity.** Clean surfaces, visible store branding, no dark patterns. If a coupon is expired, say so clearly. If a code works, show social proof. Reliability is a competitive advantage.
